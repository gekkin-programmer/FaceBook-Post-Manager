Privacy Policy & Terms of Service

Last Updated: October 22, 2025

1. Introduction

This Facebook Post Scheduler ("ABC boot ") is a private tool that allows authorized users to schedule posts on the Facebook Page La Générale du Bâtiment. It is operated by a single developer (Me).

By using this App, you agree to this Privacy Policy and Terms of Service.

2. Data We Collect

  • Message Text: The text you enter to post on Facebook.
  • Media Files: Images or videos you upload (stored temporarily).
  • Scheduled Time: The date and time you choose.
  • Facebook Page Access: We use a Page Access Token to post on your behalf (never user passwords).

We do NOT collect: emails, names, IP addresses, or analytics.

3. How We Use Your Data

  • To schedule and publish posts on the Facebook Page.
  • Media files are sent directly to Facebook and deleted from our server immediately after posting.
  • No data is shared with third parties except Facebook (via official API).

4. Data Storage & Security

The App runs on Heroku (a secure cloud platform). Sensitive keys (Facebook tokens) are stored as encrypted environment variables and never in code.

Uploaded files are processed in memory and not saved to disk. They are deleted within seconds after being sent to Facebook.

5. Facebook API Compliance

This App complies with the Facebook Platform Policy:

  • Only uses pages_manage_posts and pages_read_engagement permissions.
  • Does not store user data beyond what is needed to post.
  • Provides this public privacy policy.

6. Your Rights

You can request deletion of any scheduled post by contacting us. Since no personal data is stored, there is nothing to export or erase under GDPR/CCPA.

7. Limitation of Liability

We are not responsible for:

  • Posts that fail due to Facebook API changes or rate limits.
  • Content violating Facebook Community Standards.
  • Delays in scheduling (we aim for ±1 minute accuracy).

8. Changes to This Policy

We may update this policy. Changes will be posted here with the new "Last Updated" date.
